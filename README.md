# Flask+Mongodb+Echarts构建的web小应用

用requests小爬虫爬取到<strong>珍爱网的数据(北京地区 20-50岁)</strong>,存储到<strong>Mongodb</strong>数据库中,并通过flask展示,同时用<strong>echarts</strong>做的数据统计并展示出来
